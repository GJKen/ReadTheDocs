欢迎来到 Lumache 的文档！
===================================

** Lumache ** (/lu'make/) 是一个供厨师和美食爱好者使用的 Python 库
创建混合随机成分的食谱。
它从`Open Food Facts 数据库<https://world.openfoodfacts.org/>`_中提取数据
并提供 *simple* 和 *intuitive* API。

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
