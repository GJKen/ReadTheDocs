欢迎来到 Lumache 的文档！
====================================

**Lumache** (/lu'make/) 是一个面向厨师和美食爱好者的 Python 库
创建混合随机成分的食谱。
它从`Open Food Facts 数据库<https://world.openfoodfacts.org/>`_中提取数据
并提供 *simple* 和 *intuitive* API。

查看 :doc:`usage` 部分了解更多信息，包括
如何:ref:`安装`项目。

.. note::

   该项目正在积极开发中。
